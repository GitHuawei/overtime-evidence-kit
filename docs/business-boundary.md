# Public Business Boundary

本文档说明公开仓库中可以讨论的服务边界。它不是价格表，不是销售策略，不是 paid pilot 方案，也不是私有交付 SOP。

## 可以公开说明

- 项目是 mock-only 开源工具包。
- demo 使用完全虚构的数据。
- public outputs 展示 evidence package、Markdown report 和 CSV evidence index 的结构。
- 维护者未来可能提供证据结构化整理协助或咨询。
- 任何咨询都应先确认授权、隐私边界、材料范围、保留和删除规则。

## 不在公开仓库说明

- 真实价格。
- 报价策略。
- 成交记录。
- 客户 intake 原始流程。
- 私有交付步骤。
- paid pilot 细节。
- 真实客户材料。
- 法律结论、胜率判断或结果承诺。

## 用户安全提醒

不要在 GitHub issue、PR、discussion、commit、截图或附件中提交真实聊天、真实 Git、真实录音、真实身份、真实公司、真实项目、真实金额、真实地址、真实手机号、合同、工资或任何可识别真实案件的信息。

如需了解公开安全的咨询边界，请先阅读 `docs/customer-safety.md` 和 `docs/service-overview.md`。
