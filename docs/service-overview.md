# Service Overview

本文档只说明公开安全的服务咨询边界。它不是价格表，不是交付 SOP，也不是法律服务承诺。

## 可能提供的协助

维护者未来可能在私下渠道提供证据结构化整理协助或咨询，例如：

- 帮助理解如何把材料组织成事件和证据索引。
- 帮助识别材料中的结构缺口。
- 帮助设计可复核的时间线和证据目录。
- 帮助准备给专业人士进一步复核的问题清单。

## GitHub 不接收真实材料

不要在 GitHub issue、PR、discussion、commit、附件或截图中提交真实聊天、真实 Git、真实录音、真实身份、真实金额、真实合同或任何可识别真实案件的信息。

## 咨询前应先确认

- 你是否有权提供相关材料。
- 哪些材料可以被讨论，哪些必须排除。
- 是否需要律师或其他专业人士参与。
- 材料如何存储、保留和删除。
- 输出只做结构化整理还是还需要专业法律判断。

## 明确不提供

- 法律意见。
- 法律结论。
- 胜率判断。
- 协商、仲裁、诉讼或执行结果承诺。
- 公开渠道的真实证据审查。
- 真实服务流程、私有交付细节或收费交付 SOP。
