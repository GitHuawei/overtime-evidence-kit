# 客户输入清单

## 必需确认

- 材料来源是否合法。
- 是否有权使用和提交相关材料。
- 是否已删除不必要的他人隐私、客户数据、源码和商业秘密。
- 是否接受先脱敏、再整理、再复核的流程。

## 建议提供

- 脱敏聊天导出。
- 脱敏群聊导出。
- 脱敏 Git 日志。
- 发布夜或线上问题处理记录。
- 录音目录索引或脱敏摘要。
- 合同、考勤、工资流水的脱敏索引。

## 不应提供

- 真实手机号、身份证号、住址、银行卡号。
- 真实客户数据。
- 源码或商业秘密。
- 未经授权的他人隐私。
- 要求伪造、篡改或补写的材料。

## 复核问题

- 哪些日期需要纳入？
- 哪些候选事件应排除？
- 哪些事件需要律师进一步判断？
- 哪些材料仍需补充？
